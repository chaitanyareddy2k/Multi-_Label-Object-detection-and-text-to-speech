# Multi-Label Object Detection and Text-to-Speech

A complete Computer Vision project that combines **multi-label object detection** with **text-to-speech (TTS)** capabilities.  
This repository implements a multi-label detection model using **PyTorch and ResNet-50**, and converts the detected objects into human-readable speech using text-to-speech synthesis.
